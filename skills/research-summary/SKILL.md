# Research Summary

Produce a tight, decision-ready brief from a set of sources. The goal is signal, not coverage.

## When to use
A task asks you to research a topic, compare options, or boil a pile of links / documents / notes down into something someone can act on.

## Steps
1. **Gather** the sources you were given. If you're allowed to search, add 2–4 more high-quality ones. Record each source's title and link.
2. **Read for signal.** Pull only the claims that would change a decision: facts, numbers, trade-offs, risks, and direct quotes that matter. Ignore filler.
3. **Cross-check.** Where two sources disagree, say so. Anything you could find in only one place is unverified — label it.
4. **Write the brief** in this shape:
   - **Bottom line** — 1–3 sentences: what someone should take away.
   - **Key findings** — short bullets, each tied to a source.
   - **Open questions / uncertainty** — what's unclear or unverified.
   - **Recommended next step** — the single most useful action.
   - **Sources** — title + link for each.
5. **Stay honest.** No padding, no false confidence. If the evidence is thin, say it's thin.

## Output rules
- Lead with the bottom line; put detail below it.
- Every load-bearing claim cites a source.
- Prefer primary sources over aggregators.
- Keep it short enough to read in under a minute.
