Keyboard Shortcuts

The ones that save the most time. Mac + Windows/Linux.
Essentials — Copy, Paste, Undo
CmdCtrl+C
Copy selected text or file
CmdCtrl+X
Cut (copy + delete original)
CmdCtrl+V
Paste
CmdCtrl+A
Select all
CmdCtrl+Z
Undo
Cmd+ShiftCtrl+Shift+Z
Redo (undo the undo)
Screenshots
Cmd+ShiftWin+Shift+5S
Screenshot toolbar — THE ONE. Area select, window, screen, record. This is the good one.
Cmd+ShiftWin+4PrtSc
Screenshot area select — quick crosshair, drag to capture a region
Cmd+ShiftWin+Shift+3S
Screenshot entire screen Mac: saves to desktop
Cmd+Shift+4Alt+PrtSc Space
Screenshot a specific window Mac: hit Space after Cmd+Shift+4, then click window
Text Editing — Faster Than You Think
Shift+Enter
New line without sending — works in Claude, Slack, most chat apps
OptionCtrl+Left/Right
Jump one word at a time — hold Shift to select word by word
CmdHome/End+Left/Right
Jump to start/end of line
CmdCtrl+Up/DownHome/End
Jump to start/end of document
CmdCtrl+DeleteBackspace
Delete entire line (to left of cursor)
OptionCtrl+Delete
Delete one word at a time
CmdCtrl+F
Find on page — works in browsers, docs, most apps
Finder (Mac) / File Explorer (Windows)
Cmd+ShiftWin+GR
Go to folder by path Mac: paste a file path, hit Enter. Win: type path in Run dialog.
Cmd+ShiftCtrl+Shift+N
New folder
Space
Quick Look / Preview Mac only — preview any file without opening it
CmdF2+Enter
Rename selected file Mac: select file, press Enter. Win: select file, press F2.
Cmd+DeleteDelete
Move to Trash / Recycle Bin
Browser
CmdCtrl+T
New tab
CmdCtrl+W
Close current tab
Cmd+ShiftCtrl+Shift+T
Reopen last closed tab — lifesaver when you accidentally close something
CmdCtrl+L
Jump to address bar — start typing a URL or search immediately
Cmd+OptionCtrl+Left/RightTab / Ctrl+Shift+Tab
Switch between tabs
CmdCtrl+R
Refresh page
Cmd+ShiftCtrl+Shift+R
Hard refresh (bypass cache) — when a page won't update after changes
Window Management
CmdAlt+Tab
Switch between apps
Cmd+`Alt+Esc
Switch between windows of same app Mac only with backtick. Huge for multiple Finder windows.
Ctrl+Left/RightWin+Left/Right
Switch desktops (Mac) / Snap window (Win)
Cmd+HWin+Down
Hide / minimize current window
System
Cmd+SpaceWin
Spotlight / Start menu search — fastest way to open any app, file, or setting
Cmd+Option+EscCtrl+Shift+Esc
Force quit / Task manager — when an app freezes
Cmd+,
Open preferences for current app Mac only — works in almost every app
Terminal
Ctrl+C
Cancel running command — different from Cmd+C copy
Up Arrow
Previous command — scroll through command history
Tab
Autocomplete file/folder names — type first few letters, hit Tab
Ctrl+L
Clear terminal screen
