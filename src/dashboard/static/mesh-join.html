If the button does nothing (common)
  1. Install the Tailscale app (App Store / Play Store).
  2. If you are already on another Tailscale network, log out of it first - this is a separate network.
  3. Account menu -> Use custom coordination server / Use an alternate server.
  4. Enter the coordinator above, then sign in with the join code.
  5. When the phone is connected, open your Cove in the browser and sign in as you. Leave MagicDNS / accept DNS on when asked.

Join codes are single-use and expire in about an hour. Generate a new QR from your Cove if this one is stale. After join, give chat a minute if names are still catching up (relay before direct is normal).
