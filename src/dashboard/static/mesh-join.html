Name this device (#MESH-NAME)

Junk names like localhost-… or invalid-… make family device lists hard to read. Before or right after joining:

  1. Give the phone a clear name in system settings (e.g. jade-iphone) when the OS allows it — Tailscale often picks that up on join.
  2. On a computer already on the mesh you can rename later: tailscale set --hostname your-device-name

Join codes are single-use and expire in about an hour. Generate a new QR from your Cove if this one is stale. After join, give chat a minute if names are still catching up (relay before direct is normal).
